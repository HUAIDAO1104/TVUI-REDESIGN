tvOS 26 Liquid Glass 核心设计原则
① 光物理模拟 — 玻璃折射感来自顶部 1px 高光 + 渐变内阴影，非真实 blur 也能呈现  ② 深度分层 — 背景 0%→ 玻璃层 15%→ 内容层 100% 三层不透明度  ③ 焦点即动效 — 聚焦 = 白色外环 + 内部高光强化 + 轻微放大 scale(1.04)

  • 背景图向左渐变为纯色（gradient 70%→100%），保证文字可读
  • 海报 artwork 作为右侧装饰，不遮盖文字区域

  • 未聚焦卡片：opacity 0.7 + 无边框 → 视觉降噪
  • 卡片比例 2:3 竖图（电影海报标准）
  • 行标题左对齐，字号约 14sp，颜色 text_secondary

玻璃参考
动态颜色提取
专辑封面颜色渗透进背景，blur 形成的渐变光晕与封面色调完全一致。在 Android 可用 Palette API 提取主色后动态设置背景渐变。
▶
Glass Depth
多层玻璃深度
外层 3%→ 中层 5%→ 内层 8% 透明度递增，配合顶部线性高光，模拟玻璃厚度感。每层加 1px 边框，越内越亮。
渐变边框示例
发光边框聚焦
普通边框静止
边框光学系统
静止 1px 低亮度边框 → 聚焦时转为彩色发光边框 + box-shadow 外发光。渐变边框（CSS gradient border trick）模拟金属质感。

🔍 跨平台高颜值设计共通规律

🌑
深黑底色
所有顶级流媒体均使用接近纯黑的背景（#0A0A0F~#121218）。比 Material Design 推荐的 #121212 更黑，减少光晕，OLED 屏省电。
TVBox: bg_screen #0A0A0F ✅
🔲
一致圆角
卡片 12dp、弹窗/面板 16-20dp、按钮 8-10dp、标签 24dp（胶囊型）。圆角语义化命名，整体视觉"呼吸感"统一。
TVBox: radius_sm/md/lg/xl ✅
✨
克制高亮
品牌色仅用于 CTA 按钮、进度条、选中指示器，占屏面积 <5%。其余全黑白灰层级。"少即是多"的色彩哲学。
TVBox: brand_primary #E50914 ✅
🎯
焦点即动效
TV 遥控焦点切换是核心交互。聚焦动效包含：缩放 1.04-1.08x + 亮边框出现 + 可选的外发光。时长 120-180ms，缓动 decelerate。
TVBox: focus_glass_default ✅
📐
安全区边距
TV 屏 overscan 约 5%。Netflix/Apple TV 均留 48dp 水平安全边距，垂直 32dp。避免内容贴边被切割。
TVBox: tv_safe_margin_h 48dp ✅
🔤
字号层级
TV 最小可读 14sp（3m 观看距离），主体内容 16-18sp，标题 24-32sp，Hero 标题 40-56sp。切勿使用 96sp 等超大值导致截断。
TVBox: 96sp→48sp 已修正 ✅
🌊
渐变叠加
图片卡片底部渐变 transparent→#E6000000（angle 270°），保证文字在任何封面上可读，无需固定背景色。所有流媒体平台标配。
TVBox: bg_card_gradient.xml ✅
💎
分隔线哲学
用空白（spacing）代替分隔线，必须使用时 1dp + rgba(255,255,255,0.08) 极低对比度。tvOS 26 几乎完全去除了分隔线，靠圆角和间距区分区域。
TVBox: divider #1A1A2E 1dp ✅
